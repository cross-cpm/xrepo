# xrepo

xrepo is a multi-repo management for a large project

## 快速开始

### 安装

```shell
go get -u -v github.com/cross-cpm/xrepo
```

### 使用

```shell
xrepo help
```

```shell
xrepo checkout
```

## 更多命令

```shell
xrepo pull
```

```shell
xrepo push
```

```shell
xrepo status
```

```shell
xrepo rev list
```

```shell
xrepo rev save
```
