# xrepo

xrepo is a multi-repo management for a large project

## Getting Started

### Basic Installation

#### via wget

```shell
sh -c "$(wget -O- https://raw.githubusercontent.com/cross-cpm/xrepo/master/install.sh)"
```

## Using xrepo

### update

```shell
xrepo up
```

### status

```shell
xrepo status
```

### save

```shell
xrepo save
```
