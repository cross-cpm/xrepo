xrepo is a multi-repo management for a large project

## Quick Start Guide

### Install

```shell
go get -u -v github.com/cross-cpm/xrepo
```

### Usage

```shell
xrepo help
```

```shell
xrepo checkout
```

## More command

```shell
xrepo pull
```

```shell
xrepo push
```

```shell
xrepo status
```

```shell
xrepo rev list
```

```shell
xrepo rev save
```
